# Maintenance Module

## Overview
The **Maintenance Module** manages maintenance requests and work orders. It ensures smooth equipment maintenance and reduces downtime.

## Technologies Used
- **Spring Boot**
- **Spring Data JPA**
- **MySQL**
- **Kafka** (Message Queue for Work Orders)
