500- INTERNAL SERVER ERROR
Oops! Something went wrong on our end.
A web server may return a 500 Internal Server Error HTTP status code when it encounters an unexpected condition that prevents it from fulfilling the request. This typically occurs when there is a misconfiguration on the server, a bug in the server-side code, or an issue with the server's resources or dependencies. Status code 500 responses are the result of the server being unable to process the request due to an internal issue. Please try again later. If the issue persists, contact support.
